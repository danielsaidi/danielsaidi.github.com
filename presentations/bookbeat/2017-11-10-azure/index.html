# Notification Hubs
## Varför Notification Hubs? * Flera PNS (Platform Notification Systems) * **APNS** - Apple Push Notification Service * **FCM** - Firebase Cloude Messaging * **WNS** - Windows Notification Service
![PNS](notifications_pns.png)
## Varför? (forts.) * PNS:er har inga gemensamma gränssnitt * Azure Notification Hubs löser detta * Plattformsoberoende PNS-proxy * Pusha enkelt med ett standardiserat API
![PNS](notifications_hub.png)
## Features * Skräddarsy och rikta utskick * Segmentera på aktivitet, intressen etc. * Cross platforms, cross backend * Telemetri/tracking - Vad har skickats, hur har det använts?
## Pris * 1 miljon notifikationer gratis(?) * Tre nivåer - Free, Basic, Standard * Nivåbaserade antal utskick/enheter * Schemalagda notiser, SLA etc. endast tillgängliga i Standard.
