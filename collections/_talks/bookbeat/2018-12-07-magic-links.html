# BookBeat ## Magic Links
## Varför? * Bevara inloggning * App till webb * Webb till app * Länk till app (t.ex. i e-post)
## Exempel * App: Kontoinställningar * Webb: Öppna i appen * E-post: Signup / Kom igång

Flöde - app till webb

  • 📱 → url → ☁️
  • 📱 ← magic link ← ☁️
  • 📱 → öppna magic link → 🌎
  • 🌎 → logga in med magic link → ☁️
  • 🌎 ← inloggningsresultat ← ☁️

Flöde - webb till app

  • 🌎 → deep link → ☁️
  • 🌎 ← magic link ← ☁️
  • 🌎 → öppna deep link → 📱
  • 📱 → logga in med magic link → ☁️
  • 📱 ← inloggningsresultat ← ☁️

Flöde - mejl till app

  • ☁️ → deep link → ✉️
  • ✉️ → öppna deep link → 📱
  • 📱 → logga in med magic link → ☁️
  • 📱 ← inloggningsresultat ← ☁️
## Nuläge * En länk i appen klar * Alla BookBeat-länkar ska få stöd * Webb/mejl kommer senare * Appen bygger stöd i förväg
